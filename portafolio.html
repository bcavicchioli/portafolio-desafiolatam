Acerca de mí

Ingeniero Civil Industrial con experiencia en transformación digital, automatización de procesos y desarrollo de soluciones tecnológicas, enfocado en resolver problemáticas operacionales y diseñar proyectos de alto impacto para distintas organizaciones públicas y privadas. A lo largo de su trayectoria, ha desempeñado diversos roles estratégicos y técnicos, tales como analista de control de gestión, analista de procesos, ingeniero de procesos tecnológicos, Product Owner, jefe de proyectos y líder de transformación digital, liderando equipos multidisciplinarios en entornos complejos y altamente exigentes. Cuenta con sólidos conocimientos en levantamiento, rediseño y estandarización de procesos, así como en la implementación y gestión de normativas ISO 9001 (calidad), ISO 14001 (medio ambiente) e ISO 27001 (seguridad de la información). En el ámbito tecnológico, posee experiencia práctica en lenguajes de programación como Python, Java, JavaScript, React y Node.js, además de manejo de bases de datos relacionales y no relacionales (Oracle, PostgreSQL y MongoDB). Ha trabajado en arquitecturas modernas mediante microservicios con Spring Boot, integrando soluciones escalables y sostenibles. Se destaca por su capacidad para diseñar e implementar soluciones digitales centradas en el cliente, agregando valor mediante la automatización de tareas, la digitalización de procesos, la mejora continua, y la reestructuración organizacional, siempre con un enfoque en eficiencia, calidad y sostenibilidad.

Desafíos

GITHUB PAGES

Aplicación práctica de conocimientos en Git y GitHub mediante la configuración de un flujo de trabajo colaborativo basado en un caso real. El proyecto se publica en GitHub Pages, utilizando el portafolio como página principal y proporcionando enlaces directos a los repositorios de proyectos desarrollados previamente.

TERMINAL, GIT Y GITHUB

Aplicación de las etapas del control de versiones con Git para gestionar un repositorio local, incluyendo la realización de múltiples commits que registran el progreso del proyecto. El código versionado se publica en un repositorio remoto utilizando una conexión SSH, asegurando una sincronización eficiente y segura entre el entorno local y remoto.

BOOTSTRAP

Implementación de un sistema de grillas utilizando Bootstrap para distribuir el contenido de forma responsiva, adaptando la página web a distintos tamaños de dispositivos. Se aplican clases de Bootstrap en componentes como tarjetas (card) y formularios, guiándose por la documentación oficial para asegurar una correcta integración y presentación visual.

POSICIONAMIENTO

Aplicación de técnicas de posicionamiento en CSS mediante el uso de propiedades como display (inline, block, inline-block) y float, con el fin de organizar adecuadamente texto e imágenes dentro de la maqueta. Se resuelve el colapso de elementos flotantes utilizando métodos adecuados, asegurando una estructura visual coherente y alineada con los requerimientos del diseño.

CSS

Aplicación de conocimientos en CSS mediante el uso de hojas de estilo externas vinculadas al documento HTML, con el objetivo de unificar y estandarizar el diseño del sitio web. Se organiza la estructura de la página utilizando la carpeta assets del proyecto e incorporan propiedades como bordes, márgenes, padding y selectores de etiqueta para definir estilos consistentes y personalizados.

HTML

Diseñar un Currículum Vitae utilizando HTML, aplicando correctamente etiquetas semánticas como <p>, <h1>, <h2>, <h3>, <a>, <img>, <title> y <meta>. Implementar la estructura del proyecto siguiendo el esquema de carpetas assets revisado en clase. Asegurar el uso adecuado de la sintaxis y jerarquía de las etiquetas HTML, manteniendo una organización clara y coherente en el documento.
